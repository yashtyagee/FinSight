## Overview

FinSight is an AI-powered financial intelligence system that automates invoice processing, detects fraud, categorizes expenses, and provides real-time insights. It is designed to help businesses make smarter and faster financial decisions.

---

## Key Features

### AI Invoice Data Extraction
- Extracts structured data from invoices using OCR  
- Supports multiple invoice formats  

### Smart Expense Categorization
- Automatically classifies expenses using machine learning  
- Reduces manual effort and improves accuracy  

### Fraud & Duplicate Detection
- Detects suspicious or duplicate invoices  
- Uses anomaly detection models like Isolation Forest and Autoencoder  

### Real-time Dashboard
- Visualizes spending patterns and trends  
- Provides instant financial insights  

### Financial Report Generation
- Generate downloadable PDF reports  
- Includes summaries, analytics, and fraud alerts
